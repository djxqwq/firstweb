## 项目简介

一个基于 Vue.js 和 Node.js 的全栈 Web 应用，实现了用户管理、数据处理和可视化展示功能。该项目采用现代化的前端框架和后端技术栈，提供了完整的用户交互体验。

## 核心功能

- 👤 **用户管理**: 注册、登录、权限控制
- 📊 **数据处理**: 实时数据采集和分析
- 📈 **可视化展示**: 图表展示和数据报表
- 🔔 **消息通知**: 实时消息推送
- 📱 **响应式设计**: 支持多端访问

## 技术栈

### 前端技术
- **框架**: Vue.js 3 + Composition API
- **状态管理**: Pinia
- **UI 组件**: Element Plus
- **构建工具**: Vite
- **样式**: SCSS + CSS Modules

### 后端技术
- **运行环境**: Node.js 18+
- **框架**: Express.js
- **数据库**: MongoDB + Mongoose
- **认证**: JWT + bcrypt
- **API 文档**: Swagger

### 开发工具
- **版本控制**: Git + GitHub
- **代码规范**: ESLint + Prettier
- **测试**: Jest + Cypress
- **CI/CD**: GitHub Actions
